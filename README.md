# Игра 2048
Красноперова Елизавета Дмитриевна, K3343

### Игровой процесс
-  Сетка 4x4 генерируется динамически с помощью JavaScript
-  В начале игры на поле появляется 1-3 случайных плитки со значениями 2 или 4
-  После каждого хода появляется 1-2 новые плитки (2 или 4) в случайных свободных ячейках
-  Слияние плиток с одинаковыми числами работает во всех направлениях
-  Корректный расчет и отображение очков (сумма значений объединенных ячеек)
-  Проверка окончания игры при отсутствии возможных ходов

### Управление
-  **Десктоп**: управление стрелками на клавиатуре
-  **Мобильные устройства**:  Виртуальные кнопки (вверх, вниз, влево, вправо), управление свайпами по игровому полю
-  Кнопка "Начать заново" - сброс состояния без перезагрузки страницы
-  Кнопка "Отмена хода" (undo) - возврат на один шаг назад
-  Кнопка "Рекорды" - просмотр таблицы лидеров

### Сохранение данных
-  Автоматическое сохранение состояния игры в localStorage
-  Сохранение истории ходов для отмены
-  Таблица лидеров (топ-10) в localStorage
-  При завершении игры: инпут для имени, сохранение рекорда

### Интерфейс
-  Адаптивный дизайн для всех устройств (ноутбуки, планшеты, смартфоны)
-  Разные цвета для каждого значения плитки
-  Плавная анимация движения и слияния плиток
-  Модальное окно с таблицей рекордов
-  Оверлей при окончании игры с формой сохранения

### Использованные технологии
-  HTML
-  CSS3 (с медиазапросами для адаптивности)
- JavaScript
